# cheatsheet

- git (`g`)
  - `a` - add
  - `aa` - add --all
  - `amen` - amend
  - `bd` - branch -D
  - `c` - commit
  - `ca` - commit (all)
  - `ch` - checkout
  - `chb` - checkout (new branch)
  - `cp` - cherry-pick
  - `d` - diff
  - `fa` - fetch (all)
  - `l` - log (pretty, oneline)
  - `pl` - pull
  - `ps` - push
  - `psu` - push to new origin
  - `r` - rebase
  - `s` - status
